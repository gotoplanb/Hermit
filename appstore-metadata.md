## Description (4000 chars max)

Hermit is a focused SSH client built for developers who work with Claude Code on remote machines. Connect to your dev server, attach to tmux sessions, and interact using configurable soft keys and voice input.

Unlike general-purpose terminal apps, Hermit is designed around a specific workflow: you have Claude Code running in a tmux session on a remote machine, and you want to monitor and interact with it from your phone. Five soft keys — 1, 2, Enter, Escape, and a microphone — give you everything you need to approve tool calls, reject changes, send commands, and dictate instructions.

KEY FEATURES

- SSH connections with ed25519 private key authentication
- Automatic tmux session attach on connect
- Configurable soft key ribbon bar optimized for Claude Code workflows
- Voice input with iOS dictation — speak your instructions, proofread, and send
- xterm.js terminal rendering with proper color and escape code support
- Automatic iCloud Drive sync — your hosts and sessions available on all devices
- Dark mode interface designed for terminal use
- Private keys stored securely in the iOS Keychain

BUILT FOR CLAUDE CODE

The default soft keys map directly to Claude Code's approval workflow:
- 1 — Accept (approve tool calls and edits)
- 2 — Reject
- Enter — Confirm multi-line input
- Escape — Cancel or interrupt
- Mic — Open voice input for longer instructions

OPEN SOURCE

Hermit is open source so you can review exactly what runs on your device and how your SSH credentials are handled. No analytics, no tracking, no data collection.
